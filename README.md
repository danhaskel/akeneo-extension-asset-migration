## Project Overview

This React + Vite application provides a help desk issue form. It is built with shadcn components.

### Key Features

- **Issue Submission Form**: Allows users to submit help desk tickets.
- **Form Fields**: Includes fields for user information, issue description, and priority.

### Libraries and Components

- **UI Components**: The form is built using `shadcn/ui`, which provides a set of accessible and customizable components.
- **Icons**: Icons throughout the application are provided by `lucide-react`, a lightweight and flexible icon library.

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
